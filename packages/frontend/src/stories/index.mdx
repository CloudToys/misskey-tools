# Tools Design System

Misskey Tools フロントエンドで使用するデザインシステムです。

## 目的

システムに一貫性を持たせるためには、各画面のデザインが統一されている必要があります。

Misskey Tools はオープンソースであり、あらゆる方からのコントリビューションを歓迎しています。

本デザインシステムは、本システムの開発に貢献する全ての方に対し、一貫したデザイン哲学・規約を提供することを目的としています。

## 周辺技術

* [Radix UI](https://www.radix-ui.com)
  * ヘッドレスUIライブラリです。実装が難しいコンポーネントの機能だけを提供し、自分でスタイルを反映して使うライブラリです。
* [Radix Colors](https://www.radix-ui.com/docs/colors)
  * カラーパレット ライブラリです。
* [Stitches](https://stitches.dev)
  * 軽量・高速・高機能で、ランタイムが存在しないCSS in JSライブラリです。
  * 本デザインシステムのコンポーネント作成にて全体的に使用している必携技術です。
* [Storybook](https://storybook.js.org)
  * このドキュメントに用いているツールです。
