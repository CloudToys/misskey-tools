# Misskey Tools

**Ultimate Toolkit for All Misskists.**

すべての Misskey ユーザーがきっと好きになるツールを取り揃えたアカウント管理ツール。

## サポート

Misskey Toolsは以下のバージョンのMisskeyを正式にサポートします。

* Misskey v13.x
* Meisskey v10.x
* Calckey v13.x

## ビルド

```
# 依存関係の解決
pnpm install

# アプリケーションのビルド
pnpm run build

# 実行
pnpm run start

# デバッグ用に起動
pnpm run dev

# Lint
pnpm run lint

# Test (まだ書いてない）
pnpm run test

# デザインシステム ドキュメントを起動
pnpm run storybook
```
